# Student Management System

## Overview

The Student Management System is a C programming project designed to manage student records through a menu-driven interface. It allows users to add, view, and search student information efficiently.

## Features

* Add Student Records
* View All Students
* Search Student by Roll Number
* Menu-Driven Interface
* Structured Data Storage Using `struct`

## Technologies Used

* C Programming Language
* OnlineGDB

## How It Works

1. Users can add student details including roll number, name, and marks.
2. All stored student records can be displayed.
3. Students can be searched using their roll number.
4. The program runs continuously until the user chooses to exit.

## Sample Data

* Roll Number
* Student Name
* Marks
